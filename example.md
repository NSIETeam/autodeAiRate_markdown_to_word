# 示例Markdown文档

这是一个展示Markdown基本语法的示例文件。Markdown是一种轻量级标记语言，能让您专注于内容本身，同时轻松获得格式清晰的文档[2](@ref)。

## 1. 标题示例

Markdown支持六级标题，通过在行首添加1到6个井号（`#`）来表示，井号后需加一个空格[6,9](@ref)。

# 这是一级标题（文档主标题）
## 这是二级标题（章节标题）
### 这是三级标题（小节标题）
#### 四级标题
##### 五级标题
###### 六级标题

## 2. 正文与文本格式

这是基本的段落文本。只需正常书写即可。段落之间用空行分隔，以实现换行[10](@ref)。

文本的**加粗**效果可以通过用两个星号或下划线包裹文本来实现，而*斜体*则用一个星号或下划线[8,12](@ref)。如果想表示~~删除线~~，可以使用两个波浪号[12](@ref)。

## 3. 列表

### 无序列表
- 列表项可以使用短横线（`-`）、星号（`*`）或加号（`+`）表示[1,4](@ref)。
  - 通过缩进可以创建嵌套列表。
    - 更深层级的嵌套。
- 回到上一级。

### 有序列表
1. 这是第一项。
2. 这是第二项。
   1. 嵌套的有序列表。
   2. 第二项的子项。
3. 这是第三项。

## 4. 引用块

> 这是引用块的示例。它通常用于引用他人的话语、注释或重点强调的内容[3,10](@ref)。
> 引用块内可以包含其他Markdown格式，例如**加粗**或*斜体*。
>
> > 引用块也支持嵌套[8](@ref)。
